# club-fifa2026-journal

这是一个用于归档和发布多个独立 HTML 页面的仓库。所有页面均以静态 HTML 文件形式存放，不依赖于特定域名或网站环境。

## 项目简介

本仓库旨在整理和保存一系列与主题相关的 HTML 页面。这些页面可能包含信息展示、工具页面、资料归档等内容，各自独立运行，互不依赖。仓库本身不作为活跃项目开发，主要用于记录和公开访问。

## 目录说明

```
.
├── index.html          # 仓库入口页面，可浏览全部归档页面列表
├── pages/              # 存放所有独立 HTML 页面
│   ├── page-01.html
│   ├── page-02.html
│   └── ...
└── assets/             # 页面所需的资源文件（如样式、图片、脚本等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/` 目录是核心内容区，每个 HTML 文件代表一个独立的归档页面。
- `assets/` 目录存放页面共用或页面引用的静态资源。
- `index.html` 提供导航与简介，方便浏览所有页面。

## 页面归档说明

- 每个 HTML 页面均为独立、自包含的文件，无需后端支持。
- 页面中引用的资源（CSS、JavaScript、图片）优先使用相对路径，确保在本地或静态托管环境中均可正常访问。
- 归档页面内容可能随时间更新，但不会删除已有页面，以保证链接可访问性。
